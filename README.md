# GodotSE-Editor-Edition-

A very niche file format that had me installing maya2019 multiple times just to handle them, ported to godot.
This is the Editor Edition, different from the Runtime Edition, this version works well only as a plugin to import models on the editor and save models on the editor. If you want to parse SE files on runtime, or don't want it to work through a addon pipeline, i suggest you take a look at the Runtime Edition instead.

# GodotSE-Editor-Edition-

A very niche file format that had me installing maya2019 multiple times just to handle them, ported to godot.

This is the **Editor Edition**, different from the **Runtime Edition**, this version works well only as a plugin to import models and animations on the editor and save models and animations from the editor. If you want to parse SE files on runtime, or don't want it to work through a addon pipeline, i suggest you take a look at the **Runtime Edition** instead.

# Instalation

Have a copy of Godot on your hands. This addon was scripted using Godot 4.5, but any version from 4.0 to 4.5 should work just fine.
Download the lastest release, and then unzip the folder on your godot project's addon folder. After that, you need to enable the plugin by navigating to Project/Plugins. If **GodotSe is unchecked, check it.**
